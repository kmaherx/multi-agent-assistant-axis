About this experiment

Two Llama-3.1-8B-Instruct agents in a solver → critic → revision loop on GSM8K, each steered along Butanium's assistant-axis at independent coefficients (α_solver, α_critic).

  • Solver pass: agent A answers (steered at α_solver).
  • Critic pass: agent B critiques A's answer (steered at α_critic).
  • Revision pass: agent A revises given the critique (steered at α_solver).

Final accuracy is numeric exact-match on GSM8K's gold answer. Positive α → more default assistant-like; negative α → less. Steering happens at a single layer (residual stream addition).
